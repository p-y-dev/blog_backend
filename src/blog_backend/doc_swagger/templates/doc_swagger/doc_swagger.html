Аутентифицированные запросы

Некоторые методы API закрыты, запрос к ним должен содержать заголовок AUTH, в котором передается JWT, полученный при регистрации, или авторизации в системе.

Запросы, которые не требуют аутентификацию:

  • /account/login - Вход пользователя в систему;
  • /account/registration - Регистрация пользователя в системе;
  • /account/reset_password - Сброс пароля у неавторизованного пользователя;
  • /auth/confirm_email - Подтверждение почты;
  • /auth/send_confirm_email - Отправка ссылки для подтверждения почты;
